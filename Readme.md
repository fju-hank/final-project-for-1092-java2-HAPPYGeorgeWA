# 程式設計(二) 期末專題
## 張奕濤 408440537

## 專題用途
(請自行修改) 此專題有關一個簡單ATM，有存款、提款、轉帳、查看餘額、列印明細表等功能。

## 使用方式
(請自行修改) 執行程式為 com.fju.Tester，一開始會要求登入帳號密碼，存款功能請按1、提款功能請按2、轉帳功能請按3、
查看餘額請按4，修改密碼請按5、離開請按0(之後可選擇是否列印明細表)。
